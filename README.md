# Conway's Game of Life (C++ & x64 Assembly)

Высокопроизводительная реализация легендарной игры «Жизнь» Джона Конвея с визуализацией популяции в реальном времени.

## Особенности проекта
* **Два ядра симуляции (x64 Assembly):** Расчет физики мира полностью вынесен в два независимых модуля на чистом ассемблере x86-64 (MASM) с жесткой оптимизацией работы с регистрами.
* **Архитектура и ООП (C++):** Логика классов, управление жизненным циклом объектов, выделение динамической памяти в Куче и вывод матрицы на экран реализованы на C++.
* **Передача параметров:** Использовано соглашение о вызовах Microsoft x64 (параметры передаются через регистры RCX, RDX, R8, R9 и стек).
* **Аналитическая телеметрия:** Реализован одновременный вывод игрового поля и динамического графика текущей популяции. В реальном времени рассчитывается процент живой популяции для максимальной наглядности колебаний.

## Управление в игре
Игра поддерживает неблокирующий опрос клавиатуры, позволяя управлять симуляцией прямо во время выполнения:
* `Esc` или `M` — Открыть интерактивное меню (Пауза).
* `W` — Увеличить скорость симуляции (FPS).
* `S` — Уменьшить скорость симуляции (FPS).

## Игровые режимы
Через встроенное меню можно переключать режимы работы ассемблерного ядра:
1. **Classic (Borders):** Классический режим с жесткими границами поля. Фигуры разбиваются о края.
2. **Torus (Seamless):** Тороидальный мир («пончик»). Края поля замкнуты сами на себя (правый с левым, верхний с нижним). Фигуры могут перемещаться сквозь стены и летать бесконечно.

## Как запустить
1. Откройте файл `GameOfLife.sln` в Visual Studio.
2. Убедитесь, что выбрана конфигурация **x64** (Debug или Release).
3. Запустите проект (клавиша `F5`).
